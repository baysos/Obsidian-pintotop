# Changelog

## 1.0.5

- Removed runtime style element creation to comply with Obsidian community plugin review requirements.
- Removed an unnecessary file explorer view type assertion.

## 1.0.4

- Added support for pinning items in the vault root.
- Switched file explorer ordering to a runtime sort-result patch so Obsidian renders pinned items normally.
- Restored the original file explorer sorting method when the plugin unloads.
- Improved bilingual user-facing text.
- Updated documentation for Obsidian community plugin release preparation.

## 1.0.3

- Improved file explorer selector compatibility.

## 1.0.2

- Attempted root container support for CSS-based ordering.

## 1.0.1

- Added automatic Chinese/English UI text based on Obsidian language.
- Improved README and release metadata.

## 1.0.0

- Initial local version.
